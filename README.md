# ChatTTS Newer

本项目通过创建一个桌面软件，来帮助ChatTTS的新手快速入门如何使用ChatTTS实现自己的语音播报。
在它内部，集合了ChatTTS相关的服务，让使用者不需要自己在电脑上部署，就可以体验ChatTTS的效果。

## development

1. 克隆本仓库
2. 执行`yarn`安装依赖
3. 执行`yarn dev`启动开发环境
4. 其他命令
    - `yarn build` 打包UI界面
    - `yarn make-icon` 生成图标
    - `yarn make` 生成桌面程序
    - `yarn start` 本地启动桌面程序预览
